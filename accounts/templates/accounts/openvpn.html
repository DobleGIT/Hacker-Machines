¿Cómo conectarte a las máquinas?

Para poder conectarte a los retos tienes primero que descargarte el archivo de configuración VPN.
Esto es necesario para que te puedas conectar a nuestra red, en donde están todas las máquinas virtuales listas para ser hackeadas. Para poder descargar el archivo haz click en el botón que se muestra a continuación.

Linux

Como recomendación a la hora de realizar los retos te sugerimos que utilices una distribución de Linux especializada en el pentesting y hacking ético, como pueden ser Kali Linux o Parrot.
Una vez descargado con tu distribución de Linux favorita, dirígete al directorio donde está ubicado el archivo y abre una terminal, a continuación ejecuta los siguiente comandos:

  1. Descarga OpenVPN utilizando: sudo apt install openvpn
  2. Ejecuta el archivo .ovpn con el comando: sudo ./nombreArchivo.ovpn
